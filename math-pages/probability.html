Probability

Measure theoretic probability

Measure theoretic probability provides the language used to treat random phenomena in a mathematically precise way. Its basic objects are outcome spaces, sigma algebras, and probability measures, which together allow one to define events, random variables, and limiting operations on a consistent foundation.

Outcome Space
$\Omega$
The outcome space is the set of all possible results of the experiment under study. Each element $\omega \in \Omega$ represents one realization. In finite models, $\Omega$ can often be written down explicitly, while in continuous models it may be much more abstract.
Sigma Algebra
$\mathcal{F}$
A sigma algebra is a collection of subsets of $\Omega$ that is closed under complements and countable unions. It determines which subsets are measurable and therefore which events can be assigned probabilities in a consistent way.
Event
Measurable subset
An event is any set in the sigma algebra $\mathcal{F}$. Informally, it is a question one is allowed to ask about the outcome space. For example, in a coin flip, the event "heads" is the set $\{H\}$.
Probability Space
$(\Omega, \mathcal{F}, \mathbb{P})$
A probability space is the triple consisting of the outcome space, a sigma algebra, and a probability measure. The measure assigns each event a number in $[0,1]$ and satisfies $\mathbb{P}(\Omega) = 1$ together with countable additivity on disjoint sets.
Random Variable
Measurable map
A random variable is a measurable function $X : \Omega \to \mathbb{R}$, or more generally to another measurable space. Measurability guarantees that statements such as $\{X \leq a\}$ are events, which is what makes distributions and expectations well defined.
Why It Matters
A stable framework
Measure theoretic probability is the language behind conditional expectation, laws of large numbers, central limit theorems, martingales, and stochastic processes. It is the framework that keeps probability coherent when the underlying space is infinite or when limits are involved.
